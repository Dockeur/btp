## Prérequis

Assurez-vous d'avoir installé les logiciels suivants :

- Node.js 
- npm (généralement installé avec Node.js)
- installer un editeur de code de votre preferencce (vs code le cas echeant)

## Installation

1. Clonez le dépôt :

   ```bash
   git clone https://github.com/efficace-plateforme/efficace-frontend.git

2. cd efficace-frontend
3. npm install
4. creer a la racine de votre projet un fichier .env, copier et coller ceci a l'interieur 
#VITE_BACKEND_URL = http://192.168.100.72:8001/api
#VITE_BACKEND_URL = http://efficace-backend.test/api
VITE_BACKEND_URL = http://localhost:8000/api
5. executer le projet avec npm run dev
